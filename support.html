ScoreArc SAT Coach Internal Support

macOS SAT study prototype · Version 1.0

If you run into issues with saved progress, practice sessions, or AI coaching, this page covers the basics.

The AI coach is unavailable

The app can use OpenAI only when you provide your own API key. Set OPENAI_API_KEY before launch, or place a key in ~/Library/Application Support/ScoreArcSATCoachInternal/openai-api-key.txt. If no key is configured, the app falls back to its built-in SAT coaching flow.

My progress looks wrong or disappeared

Student progress is stored locally on your Mac. If local app data is removed, reset, or overwritten, the stored history may be lost. Reopening the app should restore the last saved local state in normal use.

The app is behaving unexpectedly

Quit the app fully and reopen it. If the issue continues, reinstall the app. Because this build is still a prototype, reinstalling may reset locally stored progress.
